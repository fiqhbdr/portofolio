Kebijakan Privasi MyUang

Terakhir diperbarui: 14 Juni 2026

Pendahuluan

MyUang adalah aplikasi manajemen keuangan pribadi yang dirancang untuk membantu Anda mengelola uang dengan lebih baik. Kami sangat menghargai privasi Anda dan berkomitmen untuk melindungi data pribadi Anda.

PENTING: MyUang TIDAK mengumpulkan, mengirim, atau membagikan data pribadi Anda ke server eksternal. Semua data tersimpan secara lokal di perangkat Anda.

Data yang Kami Kumpulkan

MyUang TIDAK mengumpulkan atau mengirim data pribadi Anda ke server manapun. Semua data berikut disimpan secara lokal di perangkat Anda:

  • Informasi profil (nama, email, nomor telepon - opsional)
  • Data transaksi keuangan
  • Data wallet/dompet
  • Data kategori
  • Pengaturan aplikasi

Bagaimana Kami Menggunakan Data

Data yang Anda masukkan hanya digunakan untuk:

  • Menampilkan informasi keuangan Anda di aplikasi
  • Membuat laporan dan analitik lokal
  • Menyediakan fitur backup/restore (ekspor/impor lokal)
  • Meningkatkan pengalaman pengguna aplikasi

Data TIDAK digunakan untuk iklan, analitik pihak ketiga, atau tujuan komersial lainnya.

Penyimpanan dan Keamanan Data

Semua data disimpan secara lokal di penyimpanan internal perangkat Anda menggunakan teknologi Hive database. Kami tidak memiliki akses ke data Anda.

Keamanan data bergantung pada keamanan perangkat Anda. Kami menyarankan:

  • Gunakan lock screen/password di perangkat
  • Backup data secara berkala menggunakan fitur ekspor
  • Jangan share file backup ke orang yang tidak dipercaya
  • Update sistem operasi perangkat secara teratur

Izin Aplikasi

MyUang TIDAK meminta izin khusus apapun dari sistem Android:

  • TIDAK ada akses internet (tidak ada koneksi ke server)
  • TIDAK ada akses lokasi
  • TIDAK ada akses kontak
  • TIDAK ada akses kamera
  • TIDAK ada akses media/foto

Aplikasi hanya menggunakan storage internal untuk menyimpan data lokal.

Tidak Ada Analytics atau Tracking

MyUang TIDAK menggunakan:

  • Google Analytics
  • Firebase Analytics
  • Crash reporting tools
  • Third-party tracking
  • Advertising IDs
  • Cookies atau tracking pixels

Hak Anda

Anda memiliki kendali penuh atas data Anda:

  • Hapus data kapan saja (uninstall app = semua data terhapus)
  • Ekspor data untuk backup pribadi
  • Import data dari backup
  • Tidak ada akun yang perlu dihapus (tidak ada sistem akun)

Data Anak-anak

Aplikasi ini tidak dirancang khusus untuk anak-anak di bawah 13 tahun. Kami tidak secara sengaja mengumpulkan informasi dari anak-anak. Karena semua data lokal, orang tua bertanggung jawab atas penggunaan aplikasi oleh anak.

Perubahan Kebijakan Privasi

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu. Perubahan akan diinformasikan melalui update aplikasi dan tanggal "Terakhir diperbarui" akan diubah.
